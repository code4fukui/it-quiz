# it-quiz

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An online quiz game that tests your knowledge of IT-related terms and concepts.

## Demo

[IT Quiz](https://code4fukui.github.io/it-quiz/)

## Features

-   Guess the IT term from its definition.
-   Each question has five multiple-choice options (one correct, four incorrect).
-   A new 10-question quiz is randomly generated each time you play.
-   Receive your score and a pass/fail message upon completion.
-   Option to retake the quiz immediately.

## Requirements

None. Runs entirely in modern web browsers.

## Usage

1.  Visit the [IT Quiz](https://code4fukui.github.io/it-quiz/) website.
2.  The quiz starts automatically. For each of the 10 questions, select the correct IT term that matches the given description.
3.  After the final question, your score is displayed in an alert.
4.  Choose to play again or exit via the confirmation prompt.

## Data

The quiz questions and answers are sourced from the [it-words](https://github.com/code4fukui/it-words) CSV dataset.
