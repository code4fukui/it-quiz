# IT Quiz

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An online quiz game that tests your knowledge of IT-related terms and concepts.

## Demo
[IT Quiz](https://code4fukui.github.io/it-quiz/)

## Features
- Randomly generated multiple-choice questions
- Option to retake the quiz
- Scoring system to track performance

## Requirements
None. The quiz runs in the browser.

## Usage
1. Open the [IT Quiz](https://code4fukui.github.io/it-quiz/) website.
2. The quiz will start automatically, presenting you with a series of questions.
3. Select the correct answer from the choices provided.
4. After completing the 10 questions, your score will be displayed.
5. If you wish to take the quiz again, click "OK" when prompted.

## Data / API
The quiz questions are sourced from the [IT Terminology](https://github.com/code4fukui/it-words) dataset.
