# IT クイズ

このリポジトリには、IT用語のクイズアプリが含まれています。オンラインで遊べるITクイズチャレンジを提供しています。

## デモ
[ITクイズ](https://code4fukui.github.io/it-quiz/)

## 機能
- ITに関する用語のクイズを出題
- 各クイズに正解と4つの選択肢を表示
- 全10問のクイズに正解するとゲームクリア
- 再挑戦可能

## データ
クイズ問題は[IT用語集](https://github.com/code4fukui/it-words)のCSVデータを使用しています。
