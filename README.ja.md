# it-quiz

IT関連の用語や概念に関する知識を試すオンラインクイズゲームです。

## デモ

[ITクイズ](https://code4fukui.github.io/it-quiz/)

## 機能

- 提示された説明（定義）からIT用語を当てます。
- 各問題は5択（正解1つ、不正解4つ）です。
- プレイするたびにランダムで10問のクイズが生成されます。
- 終了時にスコアと合否メッセージが表示されます。
- すぐにクイズをやり直すことができます。

## 要件

なし。モダンなWebブラウザ上で完全に動作します。

## 使い方

1. [ITクイズ](https://code4fukui.github.io/it-quiz/)のWebサイトにアクセスします。
2. クイズは自動的に開始されます。全10問について、提示された説明に合う正しいIT用語を選択してください。
3. 最後の問題が終わると、アラートでスコアが表示されます。
4. 確認プロンプトで、もう一度プレイするか終了するかを選択します。

## データ

クイズの問題と解答は、[it-words](https://github.com/code4fukui/it-words)のCSVデータセットを使用しています。
